# Московский зоопарк — консольное приложение (Java + Spring Boot)

## Описание
Приложение моделирует работу зоопарка: пользователь может добавлять животных и вещи, проводить ветеринарную проверку, считать суточное потребление корма и формировать отчет об инвентаризации.  
Проект реализован с использованием Java 17, Spring Boot, Gradle, JUnit 5, Mockito и JaCoCo.

## Основные идеи
- Использованы принципы SOLID и ООП.
- Интерфейс `IAlive` описывает живые сущности с параметром потребления еды (кг/сутки).
- Интерфейс `IInventory` описывает все объекты учета.
- Классы `Animal`, `Herbivore`, `Predator` реализуют животных разных типов.
- Интерфейс `VeterinaryClinic` и реализация `DefaultVeterinaryClinic` проверяют здоровье животных.
- `ZooService` управляет животными и вещами, `ReportService` создает итоговый отчет.
- `ConsoleRunner` предоставляет интерактивный интерфейс для работы с пользователем.

## Инструкция по запуску
1. Установить JDK 17 и Gradle.
2. Клонировать проект или открыть в IntelliJ IDEA.
3. Выполнить сборку:
   ```bash
   ./gradlew clean build
   ```
4. Запустить приложение:
   ```bash
   ./gradlew bootRun

   ```
5. Следовать инструкциям в консоли:
   ```
   1. Добавить животное
   2. Добавить вещь
   3. Показать отчет
   0. Выход
   ```
